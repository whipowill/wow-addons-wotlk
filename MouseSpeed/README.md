# MouseSpeed

An addon for World of Warcraft v3.3.5 WOTLK that forces a mouse speed of ``0.25``.  The default value of ``0.5`` is too damn high and changing it via macro resets every time I reload the game.

## Install

- Download this [zip](https://github.com/whipowill/wow-addon-mouse-speed/archive/master.zip) into your ``C:\\Games\WoW\Interface\Addons\MouseSpeed`` directory
